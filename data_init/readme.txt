1. The create_and_prime_database.py script creates the NYU database.
2. The “orgs”, “groups”, “events”, “student_group_subscriptions”, “students” and “checkins” tables are created.
3. The sample data (events.json, groups.json, orgs.json, students.json and student_group_sibscriptions) is loaded into the tables. 
4. May need to run pip install mysql if not already installed
5. Execute the script with python create_and_prime_database.
6. Status messages will be displayed along with a final status message indicating a success status.
